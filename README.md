# RubiksCubeSimulator
三阶魔方模拟器，可根据真实魔方图片构建魔方状态，可自动复原。  
A 3x3x3 Rubik's Cube simulator.  Build a cube's state with real shot pictures. Automatical cube solution.

## Getting Started
1. install matlab r2021a
2. run saisyc.m
